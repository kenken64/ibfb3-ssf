## Love Calculator

### Macos/Linux
```
export LOVER_API_KEY=<your api key>
export LOVER_API_HOST=<love calculator url>
export REDISHOST=<your redis host>
export REDISPORT=<your redis port no.>
export REDISUSER=<cloud username>
export REDISPASSWORD=<cloud password>

```

### Windows
```
set LOVER_API_KEY=<your api key>
set LOVER_API_HOST=<love calculator url>
```
